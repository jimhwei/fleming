YouTube Transcription With Google Cloud and JavaScript

YouTube is a great place for entertainment and more importantly, a place to watch how to jump start your car when your batteries are dead (True story). After learning JavaScript in my post-grad porgram, I decided I want to go further with what I have learned and build things. The problem is, there are so many YouTube videos out there and not enough time to watch everything.

One of my favorite channels on Youtube is TechLead. TechLead's channel is all things programming related and geeky things. His deadpan humor and robotic charm never cease to makes me laugh. Naturally, when I saw that he had a tutorial about how to transcribe YouTube videos into a text document. I knew I had to work on this project by myself so I can read the text before deciding if it is a quality video.

Advent of Code 2020

Advent of Code is an annual puzzle contest released by Eric Wastl. Each puzzle requires programming skills to solve at all different levels. Based on your puzzle solving time, you are ranked against other players in the large event. It is a great way to sharpen your skills and try different techniques to solve problem.

Each puzzle you complete will earn you stars. Each puzzle is different from the last and they can be quite challenging!
